== Description ==
IfElseNd Go Top plugin allows the visitor to easily scroll back to the top of the page.

== Installation ==

1. Click Plugins/Add New from the WordPress admin panel
1. Search for "IfElseNd Go Top" and install

-or-

1. Download the .zip package
1. Unzip into the subdirectory 'ifelsend_gotop' within your local WordPress plugins directory
1. Refresh plugin page and activate plugin
